# Fafia (CapstoneFront)

---

## 개요

Fafia는 온라인 화상 마피아 게임입니다.
웹 환경에서 친구들과 함께 같은 공간에 있는 것 처럼 얼굴을 보며 마피아 게임을 즐길 수 있습니다.

## 셋업

```bash
# 종속된 라이브러리 설치
npm install

# 서버 실행 (포트: 7000)
npm run dev
```

백엔드 설치에 관한 설명은 [여기](https://github.com/MJoon-Jung/Mafia)로.
